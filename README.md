# Multi-Class Logistic Regression from Scratch

A clean and efficient implementation of a Multi-Class Logistic Regression model from scratch using only **NumPy**.

## Features
- **Framework-Free:** Optimization, forward pass, and training loops built completely from scratch without PyTorch or TensorFlow.
- **Vectorized Computation:** Optimized matrix operations using NumPy for fast gradient computation and updates.
- **Type Annotations:** Fully documented code with explicit Python type hints for clarity and robustness.
- **Visualizations:** Tracking of cross-entropy loss and accuracy over epochs, along with a detailed confusion matrix.

## Project Structure
- `Logistic_Regression_Numpy.ipynb`: Jupyter Notebook containing the data processing, model implementation, and evaluation.
- `data/`: Folder to store the Sign Language MNIST datasets (`sign-mnist_train.csv` and `sign-mnist_test.csv`).
